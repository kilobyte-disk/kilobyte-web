# Development

Development dependencies:<br/>
- docker<br/>
- docker-compose<br/>
- ruby (for neocities cli)<br/>

Run an apache server with docker to test it over localhost.

```
docker-compose up --build
```

And then, open up ```http://localhost:8080/``` in your favourite web browser!
